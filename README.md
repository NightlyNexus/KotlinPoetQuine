[KotlinPoetQuine](quine/src/main/kotlin/com/nightlynexus/kotlinpoetquine/Quine.kt)
====================

A Kotlin [quine](https://en.wikipedia.org/wiki/Quine_(computing)) made with [KotlinPoet](https://github.com/square/kotlinpoet).
